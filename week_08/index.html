Module 8: Unsupervised Learning

This module explores learning from unlabeled data. Clustering algorithms group similar examples together, while dimensionality reduction finds compact representations. These techniques enable data exploration, preprocessing, and discovery of hidden structure.

Learning Objectives

  • Distinguish between supervised and unsupervised learning.
  • Apply k-means clustering and understand its limitations.
  • Explain hierarchical clustering and interpret dendrograms.
  • Perform PCA for dimensionality reduction and visualization.
  • Evaluate clustering quality using silhouette scores and elbow method.

Key Concepts

  • Clustering: Group similar data points
  • K-Means: Partition into k clusters via centroids
  • PCA: Find directions of maximum variance
  • Eigenvalues: Amount of variance explained
  • Silhouette Score: Measure cluster quality
