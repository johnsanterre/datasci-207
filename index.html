DATASCI 207: Applied Machine Learning

Course Materials

Welcome to DATASCI 207. This course covers the foundations of machine learning, from linear models through deep learning architectures. Each module includes lecture code, a glossary of key terms, self-check quizzes, and additional readings.

Modules

Module 1

Introduction and Framing

Functions, models, generalization, overfitting, train/test split, baselines
Module 2

Linear Regression and Gradient Descent

Loss functions, optimization, batch/stochastic/mini-batch gradient descent
Module 3

Feature Engineering

Scaling, encoding, feature crosses, feature selection, curse of dimensionality
Module 4

Logistic Regression

Sigmoid function, decision boundaries, classification thresholds, logistic loss
Module 5

Multiclass Classification and Metrics

Softmax, cross-entropy, precision, recall, confusion matrices
Module 6

Feedforward Neural Networks

XOR problem, hidden layers, activation functions, backpropagation, optimizers
Module 7

KNN, Decision Trees, and Ensembles

Non-parametric models, entropy, information gain, random forests, boosting
Module 8

Unsupervised Learning: k-Means and PCA

Clustering, hierarchical methods, GMM, dimensionality reduction, SVD, t-SNE
Module 9

Embeddings For Text

Tokenization, one-hot, bag-of-words, word embeddings, transfer learning
Module 10

Convolutional Neural Networks

Convolutional filters, pooling, stride, padding, ImageNet, transfer learning
Module 11

Network Architecture Design

Keras Functional API, multi-input/output models, embedding layers, pipelines
Module 12

Fairness

Bias, subgroup metrics, COMPAS case study, adversarial training, ethical AI
Module 13

Advanced Topics

RNNs, seq2seq, attention mechanisms, Transformers, BERT, GPT, LLMs
