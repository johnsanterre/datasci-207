Module 1: Introduction and Framing

This module introduces the foundational concepts of machine learning: functions, models, and prediction. You will learn how to represent real-world problems as functions that map inputs to outputs, explore how such functions can be tested and evaluated, and understand the importance of generalization to new data.

Learning Objectives

  • Define key concepts in machine learning, including function, model, generalization, and overfitting.
  • Identify input/output relationships in a variety of functions and machine learning tasks.
  • Apply logical and statistical testing strategies to evaluate the performance of functions.
  • Evaluate model performance using metrics such as loss, error, and accuracy on both training and test data.
  • Formulate a machine learning problem by clearly defining inputs, outputs, and evaluation criteria.
  • Explain the purpose of baseline predictors and their role in assessing model performance.

Key Concepts

  • Function: A mapping from inputs to outputs
  • Model: A learned approximation of a function
  • Generalization: A model's ability to perform on unseen data
  • Overfitting: When a model memorizes training data but fails on new data
  • Train/Test Split: Dividing data to evaluate generalization
  • Baseline: A simple reference model for comparison

Central Concepts from Prerequisites

  • Linear algebra: functions, vectors, coordinate space
  • Basic statistics: mean, error, accuracy
