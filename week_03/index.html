Module 3: Feature Engineering

This module explores the transformation of raw data into features for machine learning. It covers multivariate regression with matrix notation, followed by key feature engineering techniques such as scaling, encoding, and selection.

Learning Objectives

  • Define multivariate linear regression using matrix notation and compute predictions.
  • Describe common feature engineering tasks (handling missing data and scaling features).
  • Apply transformations such as log scaling, z-score normalization, and bucketing.
  • Encode categorical variables and construct feature crosses.
  • Evaluate features for inclusion in a model using selection strategies and discuss the curse of dimensionality.

Key Concepts

  • Multivariate Regression: Multiple input features
  • Feature Scaling: Normalization, standardization
  • Encoding: Converting categorical variables to numbers
  • Feature Crosses: Combining features to capture interactions
  • Curse of Dimensionality: Problems with high-dimensional data
